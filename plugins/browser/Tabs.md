## Commands
### Goto a tab
`trigger: tab`    
#### Operating
- `Input` search
- `Enter` locate the selected tab

### Close a tab
`trigger: tabc`    
- `shift` supported
- `Batch operation` supported

#### Operating
- `Input` search
- `Enter` close the selected tab
    - `+ Shift` close all tabs except for tabs with red color name

### Move a tab
`trigger: tabm`
#### Operating
- `Input` search
- `query` not empty
    - `Enter` move the selected tab to index of window in `query`
